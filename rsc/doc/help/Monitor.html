Monitor GUI

The Monitor program allows the user to view a camera feed and a camera's log files. This program is password protected, requiring the user to have Access Level 3.

The main window consists of a Camera Feed area and a text browser for the Camera Log. The Program first starts with the computers local camera feed.

1. File

1.1 Save As...

This operation saves the current camera's log information from the programs text browser to a text file. It opens a file dialog for the user. The user can then choose a file name and location for the text file to be saved.

1.2 Open

The user is presented with a dialog with a list of cameras stored in the database. The user can select a camera, click ok and the main window will update. The main window camera feed updates to show the new chosen camera, the camera log text browser is also updated with the camera log information from the database.

1.3 Exit

Stops and closes the program.
