Enroll GUI

The Enroll Program allows an authorised employee (Enroller) to enroll another employee (Enrollee) into the facial recognition system. This program is password protected requirering the user to have Access Level 2.

The main window displays the system's local web camera feed and an enroll button.

Enroll Button


The enroll button first asks the enroller for the First Name and Surename of the Enrollee, then it asks for the enrollee's Employee ID. The enrollee's ID is checked so that it is unique and no one else has been enrolled with the same ID. After which the program takes images of a detected face and runs the face recognition training process on the faces in the system.
